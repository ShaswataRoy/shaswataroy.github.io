About me

Pursuing my PhD in physics from Purdue university on modelling stochastic processes inside the cell. Interested in statistical and probabilistic modelling, bayesian learning, computer vision and deep learning.

I am working with Professor Srividya Iyer Biswas and her group to analyze and model stochastic processes pertaining to cells. In particular, this involves analyzing the terabytes of data obtained from the lab and then designing probabilistic models that accurately predict scaling laws with regard to certain properties of the cell (size, shape, etc.). As far as the theoretical modelling is concerned we are working at the intersection of stochastic physics, information theory and complex systems applied to the study of bacteria cells. Apart from that, I am responsible for handling the analysis pipeline which involves:

  • Research and implement state-of-the-art computer vision models while segmenting and tracking cells. This involves working with python packages commonly used in digital image processing and implementing deep learning models that are the most promising when it comes to cell segmentation.
  • Making the analysis pipeline more efficient and easier to work with and debugging any issues that may arise so that every experiment is successfully analyzed with usable datasets as soon as possible.
  • Designing more accurate algorithms to extract cell information and visualizing this using GUIs

Hybrid Quantum Neural Network for predicting drug interaction

I was involved with the Purdue Corporate Partners program from Fall 2022 to Spring 2023. Through this program, I had worked with the Ingenii team which is a startup company aiming to solve various engineering problems using quantum computation. We implemented a quantum version of a deep learning model aimed at predicting protein-ligand binding. I was responsible for the classical part which comprised replicating a model previously developed by researchers at LLNL. This involved replicating the 3D Convolutional Neural Network and Graph Neural Network on the cluster and then finally designed a custom fully connected neural network to generate the final output. Using a custom quantum encoding this was replicated to create a quantum hybrid neural network. Simulation results demonstrate a 6% increase in prediction accuracy.

IBLab Analysis

I've developed a user-friendly graphical user interface (GUI) designed to effortlessly import experimental data, perform in-depth data analysis to extract valuable insights and statistical information, and then present these findings through interactive visualizations. This tool has been meticulously crafted with the non-technical end user in mind, particularly those individuals who conducted the experiments and may not possess coding expertise. With this GUI, users can dynamically observe how specific cell parameters evolve over time in real-time and easily manipulate these parameters to gain a deeper understanding of cell behavior, all without the need for extensive coding efforts.

Mother Machine Image Analysis

I engineered an image analysis pipeline tailored for single-cell experiments conducted within a mother machine setup, a microfluidic device. First, the input image involves image is autorotated and auto-cropped to ensure all channels are precisely orthogonal and entirely visible within the image frame. Subsequently, the channels are isolated through the utilization of the image's intensity profile. Lastly, the individual cells within each channel are delineated through the application of a neural network encoder architecture. This neural network takes the channel image as input and produces a one-hot encoding, highlighting the rows within the image where cells are situated. This involves downsampling the original image over multiple layers extracting the information , ultimately yielding an array that encodes the specific regions of the channel where cells are present.
